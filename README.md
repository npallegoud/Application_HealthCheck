# Application_HealthCheck
Application Helath Check

Download the health_check.sh file

Copy to /usr/bin

Commands:

**sudo cp health_check.sh /usr/bin/healthcheck**

**sudo chmod 777 /usr/bin/healthcheck**

From any user we can execute :

Usage:

**healthcheck**
